Chat Sheet Layout Direction

Pick the structure that feels most natural for saved chat history inside the existing large sheet.

A. History On Top, Current Chat Below
RECENT CHATS
TODAY 14:20
What changed across the last 3 meetings?
YESTERDAY
Summarize open decisions from onboarding call.
CURRENT CHAT
Can you compare action items?
Across those meetings, the unresolved items are...

Recommended

Open the sheet and immediately see history plus the active thread. Feels closest to your current idea.

B. Compact History Strip
RECENT CHATS
TODAY
Action items...
YESTERDAY
Key risks...
EARLIER
Decision recap...
CURRENT CHAT
What changed this week?
This week, the main changes were...
Anything else you want me to group by owner?

Minimal intrusion

Keeps the active chat dominant. History is faster to scan, but previews are much weaker.

C. Session Picker First
CHOOSE A SESSION
Today 14:20 · Compare open decisions
Yesterday · Summarize client concerns
Earlier this week · Pull action items
ACTIVE SESSION
Only show the full message thread after choosing one session or starting a new one.

Structured archive

Best for lots of history, but it adds one extra step before chatting and feels less immediate.
