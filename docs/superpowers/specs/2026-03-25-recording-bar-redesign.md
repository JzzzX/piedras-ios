# 录音体验重设计：「沉浸书桌」方案

**日期**: 2026-03-25
**状态**: 设计确认，待实施

## 问题

当前录音体验存在两个核心问题：
1. **折叠录音条引导性弱**：用户看不出可以点击展开，交互不直观
2. **录音界面空旷**：点击录音后只有一个录音条 + "在此书写"占位符，缺乏"在场感"

## 设计方案：「沉浸书桌」

核心理念：录音开始后，MeetingDetailView 就地"变形"进入沉浸录音模式——不跳转页面，而是隐藏无关 chrome，让笔记区 + 底部录音栏占据全部视野。通过三层视觉反馈（顶部指示器、左侧氛围线、底部波形）持续传递"正在录音"的感知。

### 1. 顶部导航栏（录音模式）

- **隐藏**：转写按钮（移入底栏）、分享按钮
- **新增**：中央实底红色 REC 徽章（`background: AppTheme.highlight`, 白色文字 `● REC`，等宽字体 10pt）
- **保留**：返回按钮、更多按钮
- **动画**：红点 opacity 脉冲 0.6→1.0→0.6，周期 1.5s
- **不显示时间戳**——时间统一在底部控制栏

### 2. 笔记区（「编辑手记」风格）

- **会议标题 + 日期**：给编辑器上下文感
  - 标题：18pt semibold
  - 日期：11pt monospace `2026.03.25 · ● REC`（REC 为红色）
- **薄分隔线**：`rgba(mutedInk, 0.12)` 1px
- **引导文字**：红色边注线（2px, `rgba(highlight, 0.15)`）+ 斜体排版
  - 内容：「随手记下关键词或想法 —— 会后 AI 将结合录音，为你整理完整笔记。」
  - 用户开始打字后淡出消失
- **底部装饰**：居中 `· · ·`（12pt, 4px letter-spacing, subtleInk 色）
- **左侧氛围线**：3px 渐变红色线条（`linear-gradient` 方向向下），opacity 0.35，录音时微妙脉动

### 3. 底部录音控制栏

**双层结构**，固定在 `safeAreaInset(edge: .bottom)`：

**第一层 — 转写预览行**：
- 左：`≡` 图标（monospace 12pt）
- 中：实时 ASR partial 文字，斜体，单行截断（`lineLimit(1)`）
- 右：`▸` 箭头
- 点击打开完整转写 Sheet（取代之前的长按手势）
- 背景：`EAE3D2` + 底部 1px 分隔线

**第二层 — 控制行**：
- 左：暂停/继续按钮，44×44 RetroSquareButton，图标 ⏸↔▶
- 中：时间 `04:20`（monospace 16pt bold）+ 波形条（WaveformView, 高度约 16pt, 8 根柱）
- 右：结束按钮，红底白字 `■ 结束`，44pt 高，2pt 黑色边框

**暂停状态变化**：
- 波形消失 → 替换为「已暂停」标签（monospace 11pt, mutedInk, 边框）
- 时间变灰（mutedInk）
- 转写行显示 "转写已暂停"（非斜体）
- 暂停按钮图标变为 ▶

### 4. 状态转换动画

**进入录音模式**（约 0.4s）：
| 时间 | 动作 |
|------|------|
| 0.0s | 顶部按钮淡出（转写、分享），REC 徽章淡入 |
| 0.1s | AI 笔记摘要 crossfade 为编辑器 + 引导文字 |
| 0.2s | 底部 CTA（笔记/提问）下滑退出，录音栏从下方滑入 |
| 0.3s | 左侧氛围线从上向下"画出" |
| 0.4s | 红点开始脉冲，波形开始跳动 |

**退出录音模式**：
- 反向播放进入动画
- 增加 0.3s "AI 正在整理..." 过渡文字（打字机效果）
- 氛围线从下向上收起

### 5. 移除旧组件

- `CollapsibleRecordingBar` 的折叠/展开逻辑不再需要——整个组件重构为底部固定栏
- 旧的 `onTapGesture` 展开交互移除
- 长按查看转写的手势移除，改为点击底栏转写行

## 设计决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| 控制栏位置 | 底部浮动 | 笔记区获得更多空间，拇指操作更自然 |
| 全屏模式 | 同页沉浸化 | 不改变页面层级，体验更连贯 |
| 视觉反馈 | 三层（顶部+左侧+底部） | 多层次传递录音感知 |
| 笔记区内容 | 简洁 + 精致引导 | 保持白纸感但不空旷 |
| 引导风格 | 编辑手记（红色边注线+斜体） | 文艺克制，与 retro 美学一致 |
| 底栏密度 | 丰富型 | 转写入口 + 波形 + 控制按钮 |
| REC 指示器 | 实底红色徽章 | 醒目，与底部红色结束按钮呼应 |
| 时间戳 | 仅底部显示 | 避免顶部+底部重复 |
